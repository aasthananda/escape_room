# Escape_Room

Overview:
Navigate your explorer through a jungle temple to reach the far-right exit. Collect treasures, avoid traps, and maneuver around rocks to maximize your score.

How to Play:
  Movement Commands:
- right / r → move right 1 space
- left / l → move left 1 space
- up / u → move up 1 space
- down / d → move down 1 space
  Jump Commands: (jump over 1 space)
- jump / jr → jump 2 spaces right
- jumpleft / jl → jump 2 spaces left
- jumpup / ju → jump 2 spaces up
- jumpdown / jd → jump 2 spaces down
  Other Commands:
- pickup / p → collect treasure if on your spot
- replay → reset the board, start over
- quit / q → end the game
- help / ? → show all commands

Scoring:
  - Treasure: +10 points
  - Trap sprung: +5 points
  - End of game (reaching exit): +10 points
  - Hit wall: -5 points
  - Move off-grid: -5 points
  - Failed trap spring: -5 points
